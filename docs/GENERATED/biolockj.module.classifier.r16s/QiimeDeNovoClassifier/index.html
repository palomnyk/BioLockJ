QiimeDeNovoClassifier#

Add to module run order:
#BioModule biolockj.module.classifier.r16s.QiimeDeNovoClassifier

Description#

Run the QIIME pick_de_novo_otus.py script on all fasta sequence files

Details#

This module runs the QIIME pick_de_novo_otus.py script on all fasta sequence files in a single script since OTUs are assigned by a clustering algorithm. Additional parameters for this script are set using exe.classifierParams. If qiime.removeChimeras = "Y", vsearch is used to find chimeric sequences in the output and the QIIME script filter_otus_from_otu_table.py is run to remove them from ./output/otu_table.biom. Inherits from QiimeClassifier.
